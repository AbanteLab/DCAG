# Model Training

- `models.py`: Baseline and DCAG model class definitions, implemented with Pyro.
- `pyro_models_training.py`: Training script for Pyro models.
- `scvi_cag.py`: Training script for scANVI from scvi package.
- `utils.py`: Utils used for data loading and performance evaluation in the training scripts.
- `pyro_models_training.sh`: Example bash script of how the benchmark for Pyro models was run to test the hyperparameter space.
- `SCANVI.sh`: Bash script of how the scANVI model was trained with different hyperparameters.
